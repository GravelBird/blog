Markdown 入门教程

March 14, 2026 · Bird&Bike

Markdown 入门教程#

Markdown 是一种轻量级标记语言，让你使用易读易写的纯文本格式编写文档，然后转换成结构化的 HTML 文档。

1. 标题#

使用 # 符号表示标题，数量越多级别越低：

1
2
3
4

# 一级标题
## 二级标题
### 三级标题
#### 四级标题

2. 段落和换行#

段落之间用空行分隔：

1
2
3

这是第一段。

这是第二段。

3. 强调#

  • 粗体：用两个星号 **粗体** 或两个下划线 __粗体__
  • 斜体：用一个星号 *斜体* 或一个下划线 _斜体_
  • 删除线：用两个波浪线 ~~删除线~~

4. 列表#

无序列表#

1
2
3

- 项目一
- 项目二
- 项目三

效果：

  • 项目一
  • 项目二
  • 项目三

有序列表#

1
2
3

1. 第一步
2. 第二步
3. 第三步

效果：

  1. 第一步
  2. 第二步
  3. 第三步

5. 链接和图片#

链接#

1

[链接文字](https://example.com)

效果：链接文字

图片#

1

![图片描述](图片地址)

6. 引用#

1
2

> 这是一段引用文字。
> 可以有多行。

效果：

这是一段引用文字。 可以有多行。

7. 代码#

行内代码#

1

使用 `print()` 函数输出内容。

效果：使用 print() 函数输出内容。

代码块#

使用三个反引号包裹代码，并可指定语言：

1
2
3
4

```python
def hello():
    print("Hello, World!")
```

效果：

1
2

def hello():
    print("Hello, World!")

8. 分隔线#

使用三个或更多的星号、减号或下划线：

1
2
3
4
5

---

***

___

效果：

9. 表格#

1
2
3
4
5

| 姓名 | 年龄 | 城市 |
|------|------|------|
| 张三 | 25   | 北京 |

10. 任务列表#

1
2
3

- [x] 已完成任务
- [ ] 未完成任务
- [ ] 另一个未完成任务

效果：

  • 已完成任务
  • 未完成任务
  • 另一个未完成任务

总结#

Markdown 语法简洁明了，适合写作、文档编写和技术博客。掌握这些基础语法，你就可以开始用 Markdown 编写精美的文档了！
